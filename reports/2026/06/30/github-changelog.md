## Claude Opus 4.8 (fast mode) is now in preview for GitHub Copilot

- 公開日 (UTC): `2026-06-29 16:47:16`
- リンク: <https://github.blog/changelog/2026-06-29-claude-opus-4-8-fast-mode-is-now-in-preview-for-github-copilot>

GitHub Copilot で Claude Opus 4.8 の **fast mode** がプレビュー提供開始。通常の Opus 4.8 と同等の知能を保ちつつ、出力トークンの生成速度を大幅に高速化したモードで、応答速度を重視する対話やコーディング用途に向く。Copilot 利用者はモデルピッカーから選択して試せる（プレビュー段階のため対象プラン・ロールアウト範囲は順次拡大）。

## Restrict issue creation to collaborators only

- 公開日 (UTC): `2026-06-29 16:08:23`
- リンク: <https://github.blog/changelog/2026-06-29-restrict-issue-creation-to-collaborators-only>

リポジトリ管理者が、Issue の作成を **書き込み権限を持つコラボレーターのみ** に制限できるようになった。誰が Issue を起票できるかをより細かく制御でき、不要・スパム的な Issue 起票を抑制したいパブリックリポジトリの運用者に有用。設定はリポジトリの設定画面から有効化する（既定動作は従来どおりで、必要なリポジトリだけ明示的に絞り込む方式）。
