## Changes to model selection for Free and Student plans

- 公開日 (UTC): `2026-06-24 16:28:36`
- リンク: <https://github.blog/changelog/2026-06-24-changes-to-model-selection-for-free-and-student-plans>

## Secret scanning adds extended metadata for Replicate secrets

## Fetch Code Quality findings via REST API

## Automatic Dependabot access to GitHub-hosted registries

- 公開日 (UTC): `2026-06-23 16:46:05`
- リンク: <https://github.blog/changelog/2026-06-23-automatic-dependabot-access-to-github-hosted-registries>

Dependabot が、GitHub Packages のプライベートレジストリに対して PAT なしでアクセスできるようになりました。GitHub-hosted のパッケージを使うリポジトリでは、依存関係更新のセットアップが簡略化されます。パッケージ設定でリポジトリへのアクセス権が付与されているか確認しておくとよいでしょう。

## Deprecation of Python 3.9 for Dependabot

- 公開日 (UTC): `2026-06-23 15:46:30`
- リンク: <https://github.blog/changelog/2026-06-23-deprecation-of-python-3-9-for-dependabot>

Dependabot が Python 3.9 のサポートを終了し、EOL になったバージョンでは依存関係更新 PR が作成されなくなる可能性があります。Python 3.9 を使い続けるリポジトリは、早めに新しい Python バージョンへ移行しておく必要があります。

## GitHub Copilot app support for BYOK

- 公開日 (UTC): `2026-06-23 08:00:02`
- リンク: <https://github.blog/changelog/2026-06-23-github-copilot-app-support-for-byok>

GitHub Copilot app が BYOK をサポートし、OpenAI や Azure OpenAI、Microsoft Foundry など自前のモデルプロバイダーを使ってエージェント実行できるようになりました。独自モデル環境を持つ組織では、推論の選択肢を広げる重要な機能です。
