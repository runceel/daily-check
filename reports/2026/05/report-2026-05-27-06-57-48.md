## 1. エグゼクティブサマリー

- **agent-framework の .NET 宣言的 Workflow で code-gen 削除 PR が登場** — [#6095](https://github.com/microsoft/agent-framework/pull/6095) は 8,443 行削除の **⚠ 破壊的変更**候補。`GenerateCode` サンプルと CodeGen 実装を消し、宣言的 Workflow の実行モデルをコード生成なしへ寄せる。
- **Python Agent Framework にファイル操作と評価面の大型追加** — [#6099](https://github.com/microsoft/agent-framework/pull/6099) が `AgentFileStore` / `FileAccessProvider`、[#6101](https://github.com/microsoft/agent-framework/pull/6101) が Foundry Adaptive Evals の rubric-generation を追加中。
- **Aspire は 13.4 リリース準備と CLI 配布まわりが活発** — [#17520](https://github.com/microsoft/aspire/pull/17520)、[#17522](https://github.com/microsoft/aspire/pull/17522)、[#17525](https://github.com/microsoft/aspire/pull/17525) が release/13.4 を安定化し、[#17517](https://github.com/microsoft/aspire/pull/17517) は stable CLI 取得元を GitHub Releases に寄せて旧配布問題を解消。
- **Azure はネットワーク運用・監視の更新が 2 件** — Virtual WAN hub を Azure Virtual Network Manager に統合する Public Preview と、VNet flow logs を Microsoft Sentinel へ送るコネクタの GA/Launched が公開された。

### 今回の差分の主要トレンド

Agent Framework は .NET 側で宣言的 Workflow の負債整理、Python 側でファイルアクセス・評価・CodeAct・A2A といった agent 実行面の拡張が同時に進んだ。Aspire は 13.4/13.5 に向けたリリースブランチ作業、CLI acquisition、TypeScript AppHost GA 後処理、Dashboard A11y が中心。ASP.NET Core は release/10.0 と Components/OpenAPI/HttpSys の修正が多く、Azure Functions Worker と dotnet/extensions は小粒ながら依存更新と既存バグの追跡が続いている。
