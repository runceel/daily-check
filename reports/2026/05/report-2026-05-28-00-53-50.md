Python 側の Declarative YAML で `AppendValue` / `EmitEvent` / `Confirmation` / `WaitForInput` または `Switch` / `Goto` を使っているワークフロー定義は **無視されて skip** (unknown-kind warning が出る)。`ConditionGroup` / `GotoAction` への置換が必要。**Python と .NET で同じ正準名で書ける**ようになるため、長期的にはクロス言語のメンテ負荷が下がる。
