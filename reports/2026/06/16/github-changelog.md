## Copilot usage metrics now include more of your active users

- 公開日 (UTC): `2026-06-15 21:30:13`
- リンク: <https://github.blog/changelog/2026-06-15-copilot-usage-metrics-now-include-more-of-your-active-users>

Copilot usage metrics が、従来のクライアント信号に加えてサーバー側テレメトリも使うようになり、Enterprise の利用状況レポートに表示されるアクティブユーザー数が増える可能性があります。
管理者側の有効化作業は不要な告知ですが、前回までの利用率やシート活用率と単純比較すると増減要因を誤読しやすいため、ダッシュボードや定期レポートの注記に反映すると安全です。
