## [In preview] Public Preview: Ingestion Volume Change dashboard in Metrics Usage Insights

- 公開日 (UTC): `2026-06-05 20:45:48`
- リンク: <https://azure.microsoft.com/updates?id=565286>

Azure Monitor の Metrics Usage Insights に、取り込み量の経時変化を比較できる **Ingestion Volume Change ダッシュボード**がパブリックプレビューとして追加されました。時系列カウントやイベント取り込みレートの急増・急落を素早く検出でき、コスト変動の調査効率化に役立ちます。Azure Monitor でコスト管理・異常検知を行うすべてのユーザーが対象で、Azure Portal の Metrics Usage Insights から今すぐ利用できます。

## [In development] Private Preview: Guest RDMA on Azure Boost

- 公開日 (UTC): `2026-06-05 20:30:12`
- リンク: <https://azure.microsoft.com/updates?id=564981>

Azure Boost を活用した **Guest RDMA** がプライベートプレビューとして開始されました（UK South リージョンから展開）。ゲスト VM 内のアプリケーションが RDMA の超低遅延・高スループットネットワークを直接利用できるようになり、HPC や AI 推論など高い通信性能が求められるワークロードに適しています。現在はプライベートプレビューのため、Microsoft へ参加申請が必要です。
