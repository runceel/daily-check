## GPT-5.2 and GPT-5.2-Codex deprecated

- 公開日 (UTC): `2026-06-05 22:32:00`
- リンク: <https://github.blog/changelog/2026-06-05-gpt-5-2-and-gpt-5-2-codex-deprecated>

2026年6月5日をもって、GitHub Copilot の全体験（Chat・インライン編集・Ask/Agent モード・コード補完）から **GPT-5.2 および GPT-5.2-Codex が非推奨**となりました。これらのモデルを明示的に選択・指定しているユーザーは、GPT-5.3 以降の後継モデルへの切り替えが必要です。既存のワークフローや設定でモデル指定を行っている場合は早急に見直してください。

## CodeQL 2.25.6 adds Swift 6.3.2 support and improves C# coverage

- 公開日 (UTC): `2026-06-05 21:30:57`
- リンク: <https://github.blog/changelog/2026-06-05-codeql-2-25-6-adds-swift-6-3-2-support-and-improves-c-coverage>

GitHub のセキュリティスキャンエンジン **CodeQL 2.25.6** がリリースされ、Swift 6.3.2 のサポート追加と C# カバレッジの強化が行われました。Swift または C# プロジェクトで code scanning を利用しているチームはスキャン精度が向上します。既存の GitHub Actions code scanning ワークフローは自動的に新バージョンへ移行されるため、追加設定は不要です。

## Enterprise-managed plugins in VS Code in public preview

- 公開日 (UTC): `2026-06-05 21:01:48`
- リンク: <https://github.blog/changelog/2026-06-05-enterprise-managed-plugins-in-vs-code-in-public-preview>

先月 Copilot CLI でパブリックプレビューとなったエンタープライズ管理プラグインが、**VS Code でもパブリックプレビュー**として提供開始されました。Enterprise 管理者が組織内の全 Copilot ユーザーに対してプラグインを一元的に設定・配布できるようになります。Enterprise 管理者は GitHub Enterprise の管理画面から VS Code 向けプラグインを構成できます。
