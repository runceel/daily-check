## [In preview] Public Preview: Application Gateway for Containers – Inference gateway

- 公開日 (UTC): `2026-06-24 19:00:43`
- リンク: <https://azure.microsoft.com/updates?id=566516>

Application Gateway for Containers が AI 向けのインファレンス ゲートウェイ機能をプレビューで追加し、Kubernetes Gateway API Inference Extension を組み込めるようになりました。コンテナベースの生成 AI / 推論ワークロードを Azure の ingress で扱う構成を検討しているユーザーにとって、将来的なルーティング設計の選択肢が広がります。現時点ではプレビュー段階のため、評価環境での動作確認を進めるとよいでしょう。

## [In preview] Generally Available: Azure NetApp Files migration assistant 

- 公開日 (UTC): `2026-06-23 20:15:55`
- リンク: <https://azure.microsoft.com/updates?id=565480>

Azure NetApp Files の移行支援機能が一般提供され、ONTAP のレプリケーション機能を活用したクラウド移行をより簡単に実施できるようになりました。オンプレミスや CVO、他クラウドから Azure NetApp Files へ移すケースで、手作業を減らしつつ移行計画を進めたいユーザーにとって有用です。NetApp/ONTAP ベースのストレージ運用をしている環境では、移行候補として検討するとよいでしょう。
