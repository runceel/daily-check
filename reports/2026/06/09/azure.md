## [Launched] Generally Available: Azure NC RTX PRO 6000 Blackwell Server Edition v6 Series Virtual Machines

- 公開日 (UTC): `2026-06-08 18:15:14`
- リンク: <https://azure.microsoft.com/updates?id=565271>

NVIDIA RTX PRO 6000 Blackwell GPU搭載の NCv6 シリーズ VM が Southeast Asia と West US 2 リージョンで GA。GPU ごとに 96GB VRAM を搭載し、AI/ML ワークロードやハイエンド 3D 処理に適している。該当リージョンで GPU 駆動アプリケーションが必要な場合は検討対象。

## [Launched] Generally Available: Premium SSD v2 disks now support non-zonal Azure Virtual Machines 

- 公開日 (UTC): `2026-06-08 18:00:23`
- リンク: <https://azure.microsoft.com/updates?id=565359>

Premium SSD v2 が非ゾーン環境（Availability Zone が未選択の単一インスタンス VM）をサポート。AZ 対応リージョンで AZ を指定せずに Premium SSD v2 を使用できるようになった。柔軟なデプロイが必要なワークロードの選択肢が拡大。

## [In preview] Public Preview: Microsoft Foundry agent security capabilities in Microsoft Defender for Cloud are transitioning to a Microsoft Agent 365 license

- 公開日 (UTC): `2026-06-08 17:15:43`
- リンク: <https://azure.microsoft.com/updates?id=565171>

**ライセンス変更が必要**: 2026-07-01 から、Microsoft Defender for Cloud の Foundry agent セキュリティ機能が Microsoft Agent 365 ライセンスへ移行される。既存 Defender ユーザーは Agent 365 への切り替えと契約確認を検討する必要がある。

## [In preview] Public Preview: Azure Site Recovery support for Linux Azure VMs with NVMe disk controllers.

- 公開日 (UTC): `2026-06-08 17:15:43`
- リンク: <https://azure.microsoft.com/updates?id=565103>

Azure Site Recovery が NVMe ディスクコントローラー搭載の Linux VM（Da/Ea/Fa v6 シリーズ、Ebsv5/Ebdsv5 など）の Azure-to-Azure 災害復旧をサポート開始。NVMe ベース VM を使用している Linux システムの障害復旧戦略がより完全になった。

## [Launched] Generally Available: Maintenance control for Azure Database for PostgreSQL - reschedule, apply on demand, view and download

- 公開日 (UTC): `2026-06-08 17:15:43`
- リンク: <https://azure.microsoft.com/updates?id=563756>

Azure Database for PostgreSQL Flexible Server の メンテナンス制御が GA。プラットフォームメンテナンスをスケジュール変更・オンデマンド実行・確認・ダウンロードできるようになり、ダウンタイムの予測と最小化が容易に。本番 PostgreSQL ユーザーは運用計画を見直す価値がある。

## [In preview] Public Preview: Azure Managed Redis now supports Entra ID based RBAC for data management 

- 公開日 (UTC): `2026-06-08 16:15:40`
- リンク: <https://azure.microsoft.com/updates?id=564873>

Azure Managed Redis が Microsoft Entra ID ベースの RBAC をサポート。共有キーに頼らず、Entra ID とリソースベースのアクセス制御で Redis データへのアクセス（読み取り・書き込み・管理）を細かく制御可能に。セキュリティ強化と権限管理の簡素化が実現。
