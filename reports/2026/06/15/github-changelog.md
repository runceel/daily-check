## GitHub Actions: Minimum version enforcement timeline for self-hosted runners

- 公開日 (UTC): `2026-06-12 16:04:57`
- リンク: <https://github.blog/changelog/2026-06-12-github-actions-minimum-version-enforcement-timeline-for-self-hosted-runners>

GitHub Actions の self-hosted runner について、github.com と GitHub Enterprise Cloud with Data Residency で最小バージョン要件の適用が再開されます。
self-hosted runner を運用している組織は、古い runner がジョブ実行不可になる前に更新計画と監視を確認する必要があります。GitHub-hosted runner のみを使うプロジェクトへの直接影響は限定的です。
