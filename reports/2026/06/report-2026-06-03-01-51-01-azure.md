### 2.1 In development

#### [In development] Private Preview: multiparty analytics with Azure Confidential Clean Rooms

- 対象サービス: Azure Confidential Computing
- 公開日 (UTC): `2026-06-02 19:30`
- リンク: <https://azure.microsoft.com/updates?id=>

Azure Confidential Clean Rooms に複数組織での **マルチパーティ分析** ワークロード向け Private Preview が追加されます。データを暗号化したまま join / 集計するシナリオで、特に広告効果測定・医療データ連携・金融リスク評価などのコラボレーションを想定。Private Preview のため、現時点では Microsoft 担当への申請ベースで段階的に公開されます。

#### [In development] Private Preview: Storage optimized Lasv5 and Laosv5 Azure VM series

- 対象サービス: Azure Virtual Machines
- 公開日 (UTC): `2026-06-02 19:30`

ローカル NVMe SSD を大量に搭載したストレージ最適化 VM シリーズ **Lasv5 / Laosv5** の Private Preview。データベース・分散ファイルシステム・分析クラスタなどで IO 帯域がボトルネックとなる場合の選択肢が追加されます。Lasv4 系列からの正常進化路線で、Spec の最終確定後に Public Preview を経て GA に進む流れ。

#### [In development] Preview: Azure Cobalt 200 Arm-based Dpsv7, Dplsv7, Epsv7, Mpsv4, and Lpsv5-series Virtual Machines

- 対象サービス: Azure Virtual Machines (Arm)
- 公開日 (UTC): `2026-06-02 21:00`

Microsoft 自社設計の Arm CPU **Cobalt 200** を搭載した次世代 VM 5 シリーズ (Dpsv7, Dplsv7, Epsv7, Mpsv4, Lpsv5) が In development。Cobalt 100 (Dpsv6/Dplsv6) からの後継世代で、Linux on Arm ワークロード (.NET, JVM, Node.js, Python の x86 → Arm 移行) のコスト効率がさらに改善する見込み。Public Preview 開始時期と SKU 階層が次回更新の注目ポイント。
