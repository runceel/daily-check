## 1. エグゼクティブサマリー

- **Microsoft Build 2026 ローンチウェーブ (Azure 全 130 件)** — 6/2 19:00 UTC を中心に発表が集中。GA 65 件・Public Preview 61 件・In development 3 件・Retirement 1 件。Microsoft Foundry / Azure Cosmos DB / Azure API Management / Logic Apps / Azure Functions のラインアップが今年の主軸 ([詳細](./report-2026-06-03-01-51-01-azure.md))
- **Microsoft Agent Framework: Declarative Workflows が安定版へ昇格** — .NET 側は [Workflows.Declarative パッケージを stable (#6254)](https://github.com/microsoft/agent-framework/pull/6254) に、Python 側は [`agent-framework-declarative` を RC (#6256)](https://github.com/microsoft/agent-framework/pull/6256) に同時昇格。YAML/JSON 宣言型ワークフローが「実験」ステータスを完全に脱した ([詳細](./report-2026-06-03-01-51-01-agent-framework.md))
- **Azure Functions の Go 言語サポート Public Preview / Serverless Agents Runtime / MCP Apps GA** — Functions が「言語ランタイム」「エージェント実行基盤」「MCP サーバホスティング」の 3 方向に同時展開。.NET worker 側は #3408 で `worker.config.json` の publish 時自動生成が修正されたのみで、機能差は当面 SDK/CLI から提供される ([詳細](./report-2026-06-03-01-51-01-azure.md))
- **Aspire 13.4.0 リリース直後のパッチ集中期間** — 13.4.1 への bump (#17819) と DCP/Redis のリグレッション修正 (#17825 / #17826 / #17827) が 6/2 に集中マージ。前回ウォッチ対象だった [#17813 (WithArgs)](https://github.com/microsoft/aspire/issues/17813) はクローズ、ただし [#17822 (Redis persistent lifetime)](https://github.com/microsoft/aspire/issues/17822) と [#17831 (Azure Storage Emulator + persistent lifetime)](https://github.com/microsoft/aspire/issues/17831) という派生リグレッションが発生中 ([詳細](./report-2026-06-03-01-51-01-aspire.md))
- **AF .NET の ChatClientFactory / Middleware 順序バグが連発報告** — [#6260](https://github.com/microsoft/agent-framework/issues/6260) / [#6262](https://github.com/microsoft/agent-framework/issues/6262) / [#6264](https://github.com/microsoft/agent-framework/issues/6264) は同一ユーザから連続起票され、`ChatClientFactory` を通すと per-request middleware の順序が逆転する/ログが 1 回しか出ない/Approval middleware が無関係関数まで承認要求するという 3 連バグ。次回サイクルの重点フォロー対象 ([詳細](./report-2026-06-03-01-51-01-agent-framework.md))

### 今回の差分の主要トレンド

今回の対象期間は **Microsoft Build 2026 (米国 6/2-3 PT) のローンチデー** と完全に重なっており、Azure 側で 130 件・GitHub 側で AF/Aspire/aspnetcore 計約 45 件の PR マージが集中した「ピーク日」となりました。テーマ別に俯瞰すると、(a) **エージェント基盤の全方位ハードニング**: Foundry Hosted Agents (Public Preview)・Voice Live GA・Agent kit for Cosmos DB GA・Foundry IQ Knowledge sources 群が一斉に出揃い、AF 側も declarative workflow を stable/RC に昇格、(b) **データプラットフォーム再編**: Azure HorizonDB (Public Preview)、Cosmos DB distributed transactions / Agentic Retrieval / Semantic reranker / Agent Memory Toolkit と「DB そのものに agentic primitive を寄せる」流れ、(c) **Aspire の VS Code 拡張テスト基盤整備** (ExTester #17772 マージ、#17727 クローズ) と 13.4.0 GA 直後のリグレッション収束フェーズ、(d) **Azure Functions の管理連携基盤化**: MCP App ホスティング GA、1400+ managed connectors Preview、Container Apps 上 Override Scale Rules GA — の 4 軸が同時進行しています。
